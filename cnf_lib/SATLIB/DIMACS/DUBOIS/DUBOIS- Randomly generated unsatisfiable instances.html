DUBOIS: Randomly generated unsatisfiable instances


Benchmark instances

Available are 13 instances, originally contributed by Olivier Dubois to the DIMACS collection, which have been randomly generated; all the instances are unsatisfiable. The generator of the instances (gensathard.c) is available at the DIMACS ftp-site in this directory.

For some overview information on the benchmark instances like the number of variables and the number of clauses we refer to this ps-file (72k) (taken from the DIMACS FTP site, updated version)
